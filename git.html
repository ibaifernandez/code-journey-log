Git Instructions


Copying a repository on local

  1. To copy this repository onto your local machine, select the SSH option and copy the line next to it.
  2. At local command line, navigate to the storage targetof the project
  3. Clone your GitHub repository onto local: git clone followed by the URL copied from GitHub

    Example:

    git clone git@github.com:USER-NAME/REPOSITORY-NAME.git.

Git Workflow


Create a file

touch FILE_NAME.FILE_EXTENSION

Example:

touch index.html

Check the status

git status
  1. A red results means that the file is not staged yet.
  2. A green results means that the file has been staged.

Adding to the stage list

  1. Staging a specific file: git add FILE_NAME.FILE_EXTENSION

    Example:

    git add index.html
  2. Staging all non-staged files: git add .

    3. For more information on the git add command, visit the official guide.


Commiting a file

git commit -m "explanatory message"


Check Git log

git log


Push Your Work to GitHub

git push origin main
